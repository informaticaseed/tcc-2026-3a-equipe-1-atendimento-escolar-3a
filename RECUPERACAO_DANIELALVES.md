## 1. O que é uma Issue e para que serve no desenvolvimento de software?

Uma Issue é um registro usado para organizar tarefas, problemas, melhorias ou dúvidas em um projeto de software. Ela ajuda a equipe a acompanhar o que precisa ser feito e quem é o responsável por cada atividade. Assim, o desenvolvimento fica mais organizado e ninguém esquece tarefas importantes.

## 2. O que é um Pull Request e por que ele é importante?

Um Pull Request é um pedido para juntar as alterações feitas em uma branch ao código principal do projeto. Antes da união, outras pessoas podem revisar o código e sugerir melhorias ou corrigir erros. Isso aumenta a qualidade do software e evita problemas no projeto.

## 3. O que é separação de camadas no código?

A separação de camadas organiza o programa para que cada parte tenha uma função específica. O arquivo `app.py` normalmente controla a aplicação e recebe as requisições, enquanto o `repositorio.py` é responsável por acessar e manipular os dados. Essa divisão facilita a manutenção, a organização e a reutilização do código.
